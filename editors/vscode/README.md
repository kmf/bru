# bru for VS Code

Two variants: **Bru Espresso** (dark) and **Bru Latte** (light).

## Install

### From source

1. Copy (or symlink) this directory into your VS Code extensions folder:

   ```bash
   # macOS / Linux
   cp -r editors/vscode ~/.vscode/extensions/bru

   # Windows
   copy editors\vscode %USERPROFILE%\.vscode\extensions\bru
   ```

2. Restart VS Code.
3. Open **Settings** > **Color Theme** and select **Bru Espresso** or **Bru Latte**.

### From the repo clone

```bash
ln -s ~/.local/share/bru/editors/vscode ~/.vscode/extensions/bru
```

Then restart VS Code and pick the theme.

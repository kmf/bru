# Bru for Zed

## Install

Copy the theme file into your Zed themes directory:

```bash
mkdir -p ~/.config/zed/themes
cp bru.json ~/.config/zed/themes/bru.json
```

Then open Zed's command palette and search for **theme selector: toggle**, or add to your `~/.config/zed/settings.json`:

```json
{
  "theme": {
    "mode": "dark",
    "dark": "Bru Espresso",
    "light": "Bru Latte"
  }
}
```

## Variants

- **Bru Espresso** — dark
- **Bru Latte** — light

Both variants are bundled in a single `bru.json` file.

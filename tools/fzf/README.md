# bru for fzf

## Install

Source the variant in your shell rc:

```bash
# ~/.zshrc or ~/.bashrc
source ~/.local/share/bru/tools/fzf/bru-espresso.sh
```

Open a new shell and run `fzf` — it'll show a bru-yellow border, bru bean prompt, and blue syntax highlighting in matches.

## Visual details

- Rounded border in bru-border color
- Label ` bru ` anchored top-left in bru-yellow
- Prompt ` ☕  ` with bean emoji
- Pointer ` ❯ ` and marker ` ✓ ` for selections
- Match highlighting in bru-blue
- Info text in bru-fg-muted

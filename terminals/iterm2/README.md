# bru for iTerm2

## Install

Double-click either `bru-espresso.itermcolors` or `bru-latte.itermcolors` to import them into iTerm2.

Or manually:

1. Open iTerm2 → **Preferences** → **Profiles** → **Colors**
2. Click **Color Presets…** → **Import…**
3. Select the `.itermcolors` file
4. Click **Color Presets…** again → choose **Bru Espresso** or **Bru Latte**
